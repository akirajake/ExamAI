# ExamAI GitHubへようこそ
ExamAIは、ブラウザ完結型の学習スケジュール管理アプリです。学校から配られた「試験範囲表（PDF）」を読み込ませるだけで、Google Gemini AIが科目ごとのバランスを考慮した最適な勉強計画を自動で作成します。

# 使い方
基本的にはGoogleAiStudioのAPIキーが必要です。また、アカウントの年齢制限などの関係もありますので保護者に頼むなどをしてください。
https://aistudio.google.com/api-keys ここから取得できます。
